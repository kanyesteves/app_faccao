# Ícones do PWA

Para gerar os ícones do PWA, você pode usar ferramentas online como:

1. **https://www.pwabuilder.com/imageGenerator** (Recomendado)
2. **https://realfavicongenerator.net/**

## Ícones necessários:

- `android-chrome-192x192.png` (192x192px)
- `android-chrome-512x512.png` (512x512px)
- `android-chrome-maskable-192x192.png` (192x192px com safe zone)
- `android-chrome-maskable-512x512.png` (512x512px com safe zone)
- `apple-touch-icon.png` (180x180px)
- `favicon.ico` (16x16, 32x32, 48x48)

## Como usar o PWA Builder:

1. Acesse: https://www.pwabuilder.com/imageGenerator
2. Faça upload de uma imagem quadrada (mínimo 512x512px) com o logo do app
3. Baixe o pacote de ícones gerado
4. Substitua os arquivos nesta pasta

## Cores do tema:
- Primary: #2f1c6a (roxo escuro)
- Secondary: #b3a2d4 (roxo claro)
